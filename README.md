# My Home and Resort Landing Page

A mobile-first, cinematic, highly animated luxury resort landing page built with React, Tailwind CSS, and Framer Motion.

## Features

- Immersive hero with parallax-style motion and floating particles
- Storytelling scroll sections: Rooms, Amenities, Experience, Gallery, Booking
- Horizontal swipe room cards with snap behavior
- Scroll-triggered animations and section transitions
- Fullscreen gallery preview modal
- Sticky mobile bottom navigation + hamburger fullscreen menu
- Scroll progress indicator and desktop section dots
- One-tap call and WhatsApp CTAs
- Data-driven placeholders for easy content/image replacement

## Tech Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide React icons

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Content customization

Edit `src/data/resortData.js` to replace images/text without changing component logic.

> Note: Current images are high-quality placeholders from Unsplash and can be swapped with final resort media anytime.
